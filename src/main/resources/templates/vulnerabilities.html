OWASP Top 10 (2021) inventory

This lab is intentionally insecure. Every endpoint below is reachable both through this UI and through the underlying /api/* JSON endpoints. Hardcoded secrets are no longer rendered here — they are sourced from environment variables at deploy time (see application.properties).

Endpoint OWASP 2021 Description
GET /api/x A01 desc

How to test from the UI

  • Comments — try posting <script>alert(1)</script> and watch it render as literal text (XSS remediation).
  • Users — non-admin users see only their own row (IDOR remediation).
  • Transfer — form posts require a CSRF token (CSRF re-enabled).
