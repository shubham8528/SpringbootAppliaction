What is this?

This is the UI for the OWASP Top 10 (2021) Vulnerability Lab. Every endpoint documented on the Vulnerabilities page is reachable both through this UI and through the underlying /api/* JSON endpoints. The two surfaces share the same Spring Security configuration, so the same authorization rules apply to both.

Form posts require a valid CSRF token. All user-controlled text rendered in the UI is HTML-escaped automatically by Thymeleaf's th:text attribute, which is the XSS remediation in action.
